# Операционные системы и архитектура компьютера
База знаний по операционным системам (Unix) и устройству ПК. 
Оформлена в виде хранилища для Obsidian, регулярно дополняется по мере изучения новых тем.
## Источники
1. Лекции курса "Операционные системы" МИЭМ НИУ ВШЭ.
2. Дополнительная литература, которая будет приложена позже в отдельном каталоге.
## Как использовать
1. Склонируйте этот репозиторий.
```bash
git clone git@github.com:bloodyEmmy/Operating-Systems.git
```
2. Откройте скачанную папку в приложении Obsidian через функцию ***"Open folder as vault"***. Это позволит корректно отображать форматирование и переходить по внутренним ссылкам между заметками.
3. В базе используется сторонний плагин ***"Spaced Repetition"*** для интервального повторения тем. При запуске Obsidian спросит о доверии плагинам - нажмите ***"Trust author and enable plugins"***.
4. Если Вы хотите использовать повторение карточек, то перейдите в настройки плагина ***"Карточки"***, найдите там строку ***"Разделитель для многострочных карточек"*** и поменяйте `?` на `***`. 
5. **Важно:** В заметках с карточками присутствуют скрытые HTML-комментарии формата `<!--SR:!2026-03-13,7,250-->`. В них хранятся мои личные интервалы повторений и статистика. Перед началом использования карточек очистите эти данные, иначе плагин будет использовать мое расписание. 
6. Чтобы автоматически удалить эти строки из всех заметок, откройте терминал в корневой папке репозитория и выполните команду:
```bash
find . -type f -name "*.md" -exec sed -i 's/<!--SR:[^>]*-->//g' {} +
```
7. ***Примечание для пользователей macOS***: команда `sed` может потребовать пустой аргумент для флага бэкапа, используйте `sed -i '' 's/<!--SR:[^>]*-->//g' {} +`.
