#flashcards 
***
Читает данные из файла (или устройства) в буфер в памяти.
- Возвращает количество прочитанных байтов.
- Не добавляет `\0` в конец строки. Если мы читаем текст и хотим работать как со строкой языка `C`, то должны сами поставить `\0` после прочитанного байта.
***
***Синтаксис.***
```c
#include <unistd.h>

ssize_t read(int fd, void *buf, size_t count);
```
***Аргументы.***
1. `fd` - [[Файловый дескриптор (FD)|файловый дескриптор]], откуда читаем => число, которое вернул [[Open системный вызов|open()]] или `0` для чтения с клавиатуры (`stdin`).
2. `buf` - куда кладем прочитанное. Заранее выделенный массив.
3. `count` - максимальное количество байтов, которое мы готовы принять.
***
***Возвращаемые значения.***
1. `> 0` - количество реально прочитанных байтов.
2. `0` - достигнут конец файла (`EOF`), читать больше нечего.
3. `-1` - ошибка.
***
Пример.
```c
ssize_t bytes_read = read(0, buffer, sizeof(buffer) - 1);
if (bytes_read > 0) {
	buffer[bytes_read] = '\0';
	printf("You wrote: %s", buffer);
}
return 0;
```
