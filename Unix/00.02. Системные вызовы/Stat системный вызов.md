#flashcards 
***
Получает метаданные файла, то есть тип (файл / каталог / ссылка), [[Права доступа к файлу|права]], владелец, размер, [[Индексный узел (inode)|inode]]-номер и тд (то есть, грубо говоря, работает с содержимым inode файлов).
***
***Есть несколько вариантов.***
1. `stat(path, &st)` - следует по [[Символические ссылки (Мягкие, Симлинк)|симлинку]] и возвращает информацию о целевом файле.
2. `lstat(path, &st)` - если это симлинк, то возвращает инфо о ***самой ссылке***.
3. `fstat(path, &st)` - по [[Файловый дескриптор (FD)|файловому дескриптору]] уже открытого файла.
***
***Синтаксис.***
```C
#include <sys/types.h>
#include <sys/stat.h>
#include <unistd.h>

int stat(const char *pathname, struct stat *buf);
int lstat(const char *pathname, struct stat *buf);
int fstat(int fd, struct stat *buf);
```
