#flashcards 
***
Удаляет имя файла ([[Жесткие ссылки|жесткую ссылку]]) из каталога.
***Принцип работы.***
1. [[Операционная система|ОС]] удаляет строку с именем `"file.txt"` из родительского каталога.
2. ОС уменьшает счетчик жестких ссылок в [[Индексный узел (inode)|inode]] на единицу.
3. Если счетчик ЖС = 0, ОС проверяет, не открыт ли этот файл программой какой-либо.
4. Если нет, то физически удаляет файл (+ удаляет inode).
5. Если да, то прячет имя файла, не трогая его данные. Файл будет удален тогда, когда программа закроет его при помощи [[Close системный вызов|close()]].
***
***Синтаксис.***
```C
#include <unistd.h>

int unlink(const char *pathname);
```
- `pathname` - строка с путем к файлу.
- Возвращает `0` или `-1`.
