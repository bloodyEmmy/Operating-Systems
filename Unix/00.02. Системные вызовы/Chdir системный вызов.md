#flashcards 
***
Системный вызов [[Ядро операционной системы|ядра]], меняющий текущий рабочий каталог [[Процесс|процесса]] (current working directory - `CWD`).
- То есть, устанавливает каталог, заданный путем `path`, в качестве текущего рабочего каталога процесса.
***
***Синтаксис.***
```C
#include <unistd.h>

int chdir(const char *path);
```
Возвращает `0` при успехе и `-1` - при ошибке.
