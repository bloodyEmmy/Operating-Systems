#flashcards 
***
Открывает файл по заданному пути и возвращает его [[Файловый дескриптор (FD)|файловый дескриптор]].
***
***Синтаксис.***
```c
#include <fcntl.h>
#include <sys/types.h>
#include <sys/stat.h>

int open(const char *pathname, int flags); // для открытия существующего файла
int open(const char *pathname, int flags, mode_t mode); // для создания нового файла
```
Обязательно пишем один из флагов (`flags`):
1. `O_RDONLY` - только чтение.
2. `O_WRONLY` - только запись.
3. `O_RDWR` - чтение и запись.
Флаг `mode` используется при создании нового файла - позволяет задать права доступа.
***
***Возвращает.***
1. `>= 0` - `FD` при успехе. Обычно это минимальный свободный номер `>=3`, так как `0-2` заняты [[Ввод, вывод, вывод ошибок|stdin, stdout, stderr]].
2. `-1` - ошибка (прим: файла нет, [[Права доступа к файлу|прав]] нет и др.).
