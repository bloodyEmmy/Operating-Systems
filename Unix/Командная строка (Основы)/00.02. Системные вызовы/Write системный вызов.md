#flashcards 
***
Используется для записи данных из буфера в файл или устройство, указанное [[Файловый дескриптор (FD)|файловым дескриптором]].
- Не ищет конец строки `\0` сам, ему надо передавать длину.
***
***Синтаксис.***
```c
#include <unistd.h>

ssize_t write(int fd, const void *buf, size_t count);
```
***Аргументы.***
1. `fd` - куда пишем. Число, которое вернул [[Open системный вызов|open()]], или [[Ввод, вывод, вывод ошибок|стандартные потоки]]:
	- `1` - стандартный вывод (`stdout`).
	- `2` - стандартная ошибка (`stderr`).
2. `buf` - указатель на массив байтов (строку), откуда брать данные.
3. `count` - сколько байтов записать.
***
***Возвращаемые значения.***
1. Количество реально записанных байтов при успехе.
2. `-1` при ошибке.
***
Пример.
```c
#include <unistd.h>
#include <string.h>

int main() {
	char *msg = "Hello from write!\n";
	write(1, msg, strlen(msg));
	return 0;
}
```
