#flashcards 
***
Создает новый [[Процесс|процесс]], копируя текущий (родительский) процесс. 
Новый называется дочерним.
***
***Синтаксис.***
```c
#include <sys/types.h>
#include <unistd.h>

pid_t fork(void); // возвращаем тип pid_t
```
Программа доходит до `fork()` и "раздваивается" на две одинаковые: родителя и ребенка. Оба процесса продолжают выполнение после `fork()`, но получают разные значения возврата.
***
***Возврат.***
1. `-1` - в родителе, если создать процесс не получилось (ошибка).
2. `0` - в дочернем, если все успешно.
3. `> 0` (`PID` ребенка) - в родительском процессе, если все успешно.
```c
pid_t pid = fork();
if (pid == -1) { /* ошибка */ }
else if (pid == 0) { /* мы в ребенке */ }
else { /* мы в родителе, pid = PID ребенка */ }
```
