#flashcards
***
Используется для [[Изменение идентичности|изменения идентичности]], т.е. смены владельца и группы файла или каталога. 
Для использования нужны привилегии [[Команда sudo|суперпользователя]].
***
***Синтаксис.***
```bash
chown <владелец>:<группа> <файл>
```
***
***Примеры.***
```bash
bob - только новый владелец
bob:users - владелец и группа
:admins - только новая группа
bob: - владелец + его группа
```
