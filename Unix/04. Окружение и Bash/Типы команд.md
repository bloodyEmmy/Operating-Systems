#flashcards
***
Для каждой команды ее тип может быть определен с помощью [[Type команда терминала|type]].
1. ***Встроенные команды*** - живут внутри [[Bash]], их не нужно искать или запускать отдельно => очень быстрые (прим: [[Cd команда терминала|cd]]).
2. ***Внешние ([[Утилита|утилиты]])*** - инструменты, которые Bash зовет в случае необходимости. Примеры: [[Ls команда терминала|ls]], [[Cp команда терминала|cp]], [[Rm команда терминала|rm]].
3. ***Скрипты*** - инструкции для выполнения в одном файле.
4. ***Алиасы (псевдонимы)*** - сокращенные слова-команды. Реализуются с помощью [[Alias команда терминала|alias]].
5. ***Функции*** - как скрипты, но живут внутри Bash.
