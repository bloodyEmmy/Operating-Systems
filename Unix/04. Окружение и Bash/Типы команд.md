#flashcards
***
Для каждой команды ее тип может быть определен с помощью [[Type команда терминала|type]].
1. ***Встроенные команды***. Живут внутри [[Bash]], их не нужно искать или запускать отдельно => очень быстрые.
2. ***Внешние (приложения)***. Инструменты, которые Bash зовет в случае необходимости. Примеры: [[Ls команда терминала|ls]], [[Cp команда терминала|cp]], [[Rm команда терминала|rm]].
3. ***Скрипты***. Инструкции для выполнения в одном файле.
4. ***Алиасы (псевдонимы)***. Сокращенные слова-команды. Реализуются с помощью [[Alias команда терминала|alias]].
5. ***Функции***. Как скрипты, но живут внутри Bash.
