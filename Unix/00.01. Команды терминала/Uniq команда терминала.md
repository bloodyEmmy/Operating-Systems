Один из [[Фильтр|фильтров]], используется часто вместе с [[Sort команда терминала|sort]]. 
***Принимает сортированный список*** данных либо со [[Ввод, вывод, вывод ошибок|стандартного ввода]], либо из файла, имя которого можно передать в аргументе.
По умолчанию удаляет повторяющиеся строки списка.
***
***Пример.***
```bash
ls /bin /usr/bin | sort | uniq | less
```
***ВАЖНО:*** есть параметр `-d` - выдает список дубликатов.
