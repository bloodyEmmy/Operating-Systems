#flashcards
***
Позволяет создать [[Файловая система|файловую систему]] на каком-то [[Раздел|разделе]] диска. 
Доступна только под правами [[Sudo команда терминала|суперпользователя]].
***Пример.***
```bash
sudo mkfs -t ext4 /dev/sdb1
```
Параметр `-t` - передача типа файловой системы.
