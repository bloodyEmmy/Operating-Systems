#flashcards
***
Ищет файлы по имени. Работает быстро, потому что ищет не по диску в реальном времени, а по заранее созданной базе [[Файловая система|файловой системы]]. Можно использовать в качестве составляющего внутри [[Конвейер|конвейера]].
Есть специальная база данных, которая по определению обновляется раз в сутки. Чтобы обновить раньше, надо использовать `updatedb` в режиме [[Sudo команда терминала|суперпользователя]].
***
***Пример.***
```bash
locate ssh
locate bin/zip
locate zip | grep bin
```
В примере использован [[Grep команда терминала|grep]].
