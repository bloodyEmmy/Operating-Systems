Как и [[Locate команда терминала|locate]], `find` выполняет поиск файлов, но только в указанном каталоге + во вложенных подкаталогах. Она сканирует весь диск, не связана с [[Файловая система|файловой системой]], но всегда видит новые файлы. 
Ищет по имени, размеру, дате, типу, [[Права доступа к файлу|правам доступа]] и тд.
- Удобно использовать вместе с [[Wc команда терминала|wc]] с параметром `-l` - кол-во строк, т.е. кол-во удовлетворяющих файлов.
- Возвращает результаты в несортированном виде.
***
***Как работает.***
1. Начинает с указанной директории, используя [[Opendir библиотечная функция|opendir()]] и [[Readdir библиотечная функция|readdir()]].
2. Для каждого найденного элемента вызывает [[Stat системный вызов|stat()]], чтобы вытащить данные из [[Индексный узел (inode)|inode]].
3. Пропускает данные через фильтры, заданные командой. Если совпало все, то выводит путь или может отдать его [[Операции в find, определяемые пользователем (exec)|-exec]].
***
***Пример.***
```bash
find playground -type f -name 'file-A'
```
Здесь используются [[Проверки в find|проверки]] + [[Предопределенные операции в find|предопределенные операции]] команды `find`.
