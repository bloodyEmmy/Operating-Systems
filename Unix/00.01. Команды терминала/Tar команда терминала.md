#flashcards
***
Используется для [[Архивирование|архивирования]].
***Синтаксис.***
```bash
tar <режим> <параметры> <путь>
```
- Есть много режимов, могут быть записаны группой по несколько вместе.
- ***Режимы пишутся строго ПЕРЕД параметрами.***
- ***Режим и параметры пишутся БЕЗ`-`***.
- ***Используется относительный путь к файлу, а не абсолютный.***
- Часто используется с [[Find команда терминала|find]] и [[Операции в find, определяемые пользователем (exec)|exec]] для поиска файлов, подлежащих включению в архив.
- Может быть частью [[Конвейер|конвейера]] вместе с [[Gzip, gunzip команда терминала|gzip / gunzip]] и [[Bzip2, bunzip2 команда терминала|bzip2 / bunzip2]].
***
***Некоторые режимы.***
1. `c` - создать архив из списка файлов и/или каталогов.
2. `x` - извлечь файлы из архива.
3. `r` - добавить указанный файл и/или каталог в конец архива.
4. `t` - выведи список содержимого архива.
***
***Популярный параметр.***
1. `f` - определяет имя архива. Если его не написать, то он будет ожидать архив из [[Ввод, вывод, вывод ошибок|стандартного ввода / вывода]]. Лучше всего использовать постоянно.
***
***Пример.***
```bash
sudo tar cf /media/BigDisk/home.tar /home
sudo tar xf /media/BigDisk/home.tar
find playground -name 'file-A' -exec tar rf playground.tar '{}' '+'
```
