#flashcards
***
Один из [[Фильтр|фильтров]], который располагает файлы по алфавиту или же ***сортирует строки текста по различным критериям***.
***
***Пример.***
```bash
ls /bin /usr/bin | sort | less
```
Здесь две директории => получаем два отсортированных списка по одному для каждого каталога + вывод с помощью [[Конвейер|конвейера]] и [[Less команда терминала|less]].
***
***Параметры.***
1. `-n` - числовое вычисление.
2. `-r` - реверс.
3. `-k` - выборочная сортировка по столбцам.
