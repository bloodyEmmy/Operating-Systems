`KILL` Убивает (т.е. завершает) [[Процесс|процесс]]. 
Нужно обладать правами [[Sudo команда терминала|суперпользователя]], чтобы посылать сигналы процессам, которыми не владеем.
***Синтаксис.***
```bash
kill <номер PID>
```
У команды есть сигналы, каждый из которых отвечает за свою работу `kill`. Выводится через `kill -l`.
***
`KILLALL` позволяет посылать сигналы нескольким процессам, соответствующим указанной программе или имени пользователя.
***Пример.***
```bash
xlogo &
xlogo &
killall xlogo
```
