#flashcards
***
Используется в [[Tar команда терминала|tar]]. Нужен для того, чтобы искать файлы / директории по шаблонам для дальнейшего извлечения из [[Архивирование|архива]].
Для этого используются [[Групповые символы (wildcards)|wildcards]], а также [[Частые классы символов]].
***
***Пример.***
```bash
tar xf ../playground2.tar --wildcards 'home/me/playground/dir-* /file-A'
```
Извлекаем только файлы, соответствующие указанному пути с групповым символом `dir-*`.
