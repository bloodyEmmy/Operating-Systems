Работает аналогично [[Бит setuid (SUID)|SUID]], только = восьмеричное число `2000` и задает [[Атрибуты файла|идентификатор группы]].
Если установить этот бит для каталога, то создаваемые внутри файлы наследуют группу этого каталога, а не группу создателя файла. Полезно, если содержимое должно быть доступно всем членам основной группы, независимо от принадлежности к группе владельца.
- Используется вместе с [[Chmod команда терминала|chmod]].
- Показатель = символ `s` при использовании [[Ls команда терминала|ls]] (снова как у `SUID`).
- Выглядит так: `drwxr-sr-x`.
***
***Проще.***
1. ***Файл***: рабочий, взявший в руки инструмент, временно получает [[Права доступа к файлу|права]] не одного конкретного человека, а всей группы, которой принадлежит этот инструмент.
2. ***Каталог***: целая бригада (группа) может редактировать чертежи каждого ее участника. То есть у рабочих Васи и Пети из одной бригады по умолчанию отдельные права и никто не может редачить чужой чертеж. `SGID` эту возможность дает - теперь каждый участник бригады может редактировать чертежи других участников этой бригады.
***
***Пример.***
```bash
chmod g+s dir
```
