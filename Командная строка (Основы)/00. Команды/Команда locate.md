#flashcards
***
Ищет файлы по имени. Работает быстро, потому что ищет не по диску в реальном времени, а по заранее созданной базе [[Файловая система]]. Можно использовать в качестве составляющего внутри [[Конвейер]].
Есть специальная база данных, которая по определению обновляется раз в сутки. Чтобы обновить раньше, надо использовать updatedb в режиме суперпользователя ([[Команда "sudo"]] или [[Команда "su"]])
***
***Пример***:
>locate ssh
>locate bin/zip
>locate zip | grep bin ([[Команда "grep"]])
