#flashcards 
***
Протокол для безопасного удаленного доступа к компьютеру. Дает зашифрованное соединение, а [[SSH Клиент|пользователь]] может дистанционно управлять [[SSH Сервер|машиной]] через [[Командная оболочка|командную оболочку]].
***
Полезно для удаленного подключения к серверу, выполнения команд на дистанционной машине, копирования файлов (прим: [[Программа rsync|rsync]]), туннелирования и тд.
***
Подключение чаще всего происходит через 2 ключа - открытый и закрытый. Открытый отправляется пользователем на сервер, сервер смотрит приватный ключ (без передачи его), если совпадает -> доступ разрешен.
***
Порт по умолчанию - `22`.
```bash
ssh user@192.168.1.100 -p 2222
```
Порт можно изменить для защиты.
